## What is CODE?

CODE is an AI-powered development system that handles planning, architecture, implementation, testing, and deployment for web apps, mobile apps, and Chrome extensions.

## Getting Started

\`\`\`bash

npm install

# Run development server
npm run dev

# Run tests
npm test

# Run tests with coverage
npm run test:coverage

# Build for production
npm run build

# Type check
tsc --noEmit

# Lint
npm run lint
\`\`\`

## Available Agents

Use these with any AI coding assistant:

| Agent | Purpose |
|-------|---------|
| `@code` | Main orchestrator - routes all requests |
| `@planner` | Requirements, scope, features |
| `@architect` | Stack selection, architecture |
| `@frontend` | UI components, state management |
| `@backend` | APIs, databases, auth |
| `@mobile` | React Native/Expo apps |
| `@extension` | Chrome extensions |
| `@tester` | Unit, integration, E2E tests |
| `@deployer` | CI/CD, deployment, monitoring |
| `@reviewer` | Code review, handoff readiness |
| `@security` | Security audit, compliance |

## Workflow Phases

1. **Planning** - `@planner` defines requirements and scope
2. **Architecture** - `@architect` selects stack and structure
3. **Implementation** - `@frontend`/`@backend`/`@mobile`/`@extension`
4. **Quality Assurance** - `@tester` and `@reviewer`
5. **Documentation** - Auto-generated by all agents
6. **Handoff** - `@reviewer` runs 30-minute rule check

## Quality Standards

- TypeScript strict mode
