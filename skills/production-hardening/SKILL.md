# Production Hardening

Loads and coordinates all 6 sub-skills based on project profile. Runs the master checklist.

## Master Checklist

### Code Quality
- [ ] All tests pass (>70% coverage)
- [ ] No TODOs, console.logs, or debug code
- [ ] Error handling covers all paths
- [ ] Input validation on every endpoint
- [ ] No hardcoded secrets
- [ ] All files under 250 lines
- [ ] Functions under 50 lines

### Infrastructure
- [ ] HTTPS enforced
- [ ] CDN configured for static assets
- [ ] Database connection pooling
- [ ] Environment variables for all config
- [ ] CI/CD pipeline operational

### Security
- [ ] OWASP Top 10 checked
- [ ] CORS configured properly
- [ ] Rate limiting in place
- [ ] Auth tokens expire
- [ ] SQL injection prevention (parameterized queries)

### Operations
- [ ] Monitoring set up
- [ ] Error tracking configured
- [ ] Backup strategy in place
- [ ] Rollback plan exists
- [ ] SLOs defined

## Usage

```yaml
# In agent config
skills:
  - production-hardening  # Master — auto-loads sub-skills by profile
```

The master skill auto-loads sub-skills based on the project profile generated by `auto-detect`.
